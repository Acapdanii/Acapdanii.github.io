Last important thing to take note is your motherboard model (H series, B series and Z series)

Motherboard Series (Important to Know)

Intel Motherboard Series:

  • H-Series – Basic/entry-level motherboards. Good for budget builds. No overclocking. Limited features.
  • B-Series – Mid-range motherboards. Better VRMs, more ports, good for gaming builds. Still no CPU overclocking.
  • Z-Series – High-end chipset. Allows CPU overclocking, better power delivery, more M.2 slots and features.
  • Q-Series – Business-oriented boards. Extra IT/management features. Not meant for gaming.

AMD Motherboard Series:

  • A-Series – Basic/entry-level (A320 / A520). No CPU overclocking. Good for simple or office builds.
  • B-Series – Most popular for gaming. Good performance, decent VRM, supports overclocking (B450 / B550 / B650).
  • X-Series – High-end chipset (X470 / X570 / X670). Best VRMs, more PCIe lanes, ideal for high-end Ryzen builds.

YOU MIGHT WANT TO CHECK:

  • Storage slot – There's a few types of storage and each one of them uses a different slot, you might want to check it before buying a motherboard.
  • Ram slot – Some motherboards come with 2 RAM slots, 4 RAM slots or even 8! You might want to check how many you need. (Average user nowadays only needs 16GB, so 2 slots is enough with a combo of 8GB+8GB.)

CPU (Processor)

Here comes the fun part, THE CPUs!!

How do you actually know if a CPU is good or not? Instead of just looking at the price, we can learn the basic naming first. Once you understand the name, you already know roughly how powerful the CPU is, which generation it is, and what it’s meant for.

Let’s start by looking at how the names are built:

Of course, I think most of you already know there are two main CPU brands: Intel and AMD Ryzen. Both of them use a similar naming idea (numbers to show how “high” the model is and what generation), but there are some small differences – you can see that from the pictures above.

I’ll just explain the important parts here.

Intel CPU Naming – i3, i5, i7, i9

For the older style like Core i3 / i5 / i7 / i9, the “i” number is the series level:

  • Core i3 – entry-level. Good for basic use: browsing, study, light office work.
  • Core i5 – mid-range. Very common for gaming + everyday use.
  • Core i7 – high-end. More cores/threads, better for heavy multitasking and high FPS gaming.
  • Core i9 – top consumer level. Lots of cores, great for streaming, editing, 3D work.

After the “i” number you’ll see digits like i5-12400 or i7-9700. The first digit (for example 9 in 9700 or 12 in 12400) tells you the generation. The rest of the numbers are the model – higher usually means slightly better within the same family.

New Intel Core Ultra Naming

Newer laptops and some desktops use names like Intel Core Ultra 7 165H. The idea is almost the same, just written differently:

  • Core Ultra – this is the new branding.
  • 7 – sub-series level (similar to the old i7: higher number = higher tier).
  • 1 in 165 – generation number.
  • 65 – model inside that generation (higher usually = better).
  • H – suffix. Tells you the type of chip (H usually means high-performance laptop CPU).

For example, a Core Ultra 5 is more like a mid-range chip, while a Core Ultra 7 is the higher tier. You can think of it just like the old i5 vs i7 idea.

AMD Ryzen Naming – Ryzen 3, 5, 7, 9

AMD Ryzen uses a very similar style: Ryzen 3 / 5 / 7 / 9.

  • Ryzen 3 – entry-level. Basic use, light gaming.
  • Ryzen 5 – mid-range. Super popular for gaming + daily use.
  • Ryzen 7 – high-end. Great for high FPS gaming and multitasking.
  • Ryzen 9 – top consumer level. Many cores, for heavy editing, streaming, and work.

The numbers after that, like Ryzen 5 5600 or Ryzen 7 7800X3D, work the same way:

  • The first digit (for example 5 in 5600 or 7 in 7800) is the generation.
  • The rest is the model – again, higher usually means better inside that series.

Threadripper (Bonus)

You might also see the name Ryzen Threadripper. This is AMD’s workstation / creator CPU line – way more cores and way more expensive. It’s not really meant for normal gaming builds, but good to know it exists.

So the next time you see a CPU like Core i5-12400 or Ryzen 7 5800X, you’ll already have an idea of where it stands: brand → series (3/5/7/9) → generation → model → suffix.

CPU SUFFIXES

CPU names sometimes end with letters. These are called suffixes, and they tell you what type of chip it is (power usage, overclocking ability, integrated graphics, etc).

Intel Desktop CPU Suffixes

  • K – Unlocked for overclocking. Higher performance.
  • F – No integrated graphics (needs a GPU).
  • T – Low-power desktop version.
  • S – Special edition / slightly boosted version.
  • X – High-End Desktop (HEDT) CPUs. For creators, workstation use. NOT meant for normal PC builders.
  • XE – Extreme Edition (very high-end, workstation-level CPUs).

⚠️ Some models combine suffixes, for example: Intel Core i5-12600KF. A combo like K + F becomes KF (unlocked + no iGPU).

Intel Laptop CPU Suffixes

  • U – Ultra-low power (thin & light laptops).
  • P – Mid-power thin laptops.
  • H – High-performance laptop CPU.
  • HX – Desktop-class performance in a laptop.
  • HK – High-performance + unlocked (rare).
  • G1 / G4 / G7 – iGPU strength indicator for older 10th/11th gen CPUs.

AMD Ryzen Desktop CPU Suffixes

  • No suffix – Normal desktop CPU.
  • X – Higher clock, higher performance.
  • XT – Slightly boosted/binned version of an X CPU.
  • G – Has integrated graphics (APU).
  • GE – APU with lower power use.
  • X3D – Has 3D V-Cache. Best gaming CPUs.

AMD Ryzen Laptop CPU Suffixes

  • U – Low-power, office & thin laptops.
  • HS – Balanced (performance + battery).
  • H – High-performance gaming laptops.
  • HX – Highest-power Ryzen laptop CPU.

Bonus: What is “G7” on Intel CPUs?

Older Intel laptop CPUs like i7-1165G7 use “G + number” to show GPU strength.

  • G1 – weakest iGPU
  • G4 – mid-level
  • G7 – strongest iGPU (Iris Xe)

So i7-1165G7 means: 11th gen CPU + the strongest integrated graphics for that generation.

WHICH SUFFIXES SHOULD BEGINNERS CARE ABOUT?

  • For Intel Desktop – Just remember: K (overclock), F (no graphics).
  • For Intel Laptop – Look for: H (gaming), U (battery life).
  • For Ryzen Desktop – Watch for: X (performance), G (has iGPU).
  • For Ryzen Laptop – H for gaming, U for battery.

Everything else is more “advanced builder knowledge”, so don’t worry too much for now.

RAM (Memory)

For RAM, it’s super simple. Every type of RAM has a unique gap (notch) on the stick that matches the slot on the motherboard. So you MUST check what type of RAM your motherboard supports.

TYPES OF RAM

  • DDR3 – Older generation RAM (usually 1333MHz – 1866MHz). Mostly found on older PCs, no longer recommended unless upgrading an old system.
  • DDR4 – Most common RAM type for builds from 2016–2023. Runs between around 2400MHz – 3600MHz for standard use. Good price and good speed.
  • DDR5 – Latest RAM standard (4800MHz – 8000MHz+). Faster and more efficient. Mostly used in modern Intel 12th–14th gen and Ryzen 7000 builds.

SO WHAT IS THE MINIMUM AMOUNT OF RAM?

Well, it depends on what you do. If all you're doing is browsing the internet, watching videos, doing assignments, light coding, or office work like PowerPoint or Word, 8GB of RAM is more than enough.

On the other hand, if you want to play games smoothly, the minimum is 16GB. Heavier games can use more than 16GB, especially modded games like Minecraft with large modpacks. AAA titles on max settings can easily use above 20GB of RAM.

DOES RAM SPEED MATTER THAT MUCH?

Yes… but not as much as you think. Yes, it matters, but not dramatically. For example, if you compare DDR5 4800MHz and DDR5 6000MHz, you might only see a small difference of around 5–10 FPS in games.

BONUS

You might see things like DDR4 3200MHz CL40. So what is CL40?

CL stands for CAS Latency — basically the number of cycles it takes between your processor asking for data from the RAM and the RAM actually delivering it.

Just think of it like how fast the RAM hands over data to the CPU.

LOWER CL VALUE = FASTER

You will see variations like CL40, CL32, CL30, CL26… And YES — CAS latency affects performance more than raw MHz speed.

For example: DDR5 4800MHz CL26 can actually perform better than DDR5 6000MHz CL40.
